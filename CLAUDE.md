# Project Rules

## Stack
- Next.js
- TypeScript
- Tailwind CSS

## Product
- Mobile-first web prototype
- Show 3 top news cards only
- Clean Apple-minimal style
- White background, generous spacing
- Serif headlines, sans-serif body text

## News Card Structure
Each card should include:
- title
- Korean quick summary
- who is involved
- why it matters
- "Say It" section

## Say It
- Tabs: A1, B1, B2, C1
- Default tab: B1
- Each level must contain 5 sentences
- Tone must sound natural and spoken
- Do not write like a newspaper article
- Do not use overly academic or formal phrasing
- The tone should feel like explaining the issue to a friend

## UI
- Minimal interactions
- No heavy animation
- Calm transitions only
- Focus on readability

## Scope
- Start with mock data only
- Do not add backend yet
- Do not add AI API yet
- Do not add authentication
